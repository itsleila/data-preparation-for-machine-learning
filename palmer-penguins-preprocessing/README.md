# Palmer Penguins – Data Preparation for Machine Learning

Este diretório contém uma coleção de exercícios práticos de **pré-processamento de dados e preparação para Machine Learning** utilizando o dataset **Palmer Penguins**.

Os exercícios abordam conceitos fundamentais de _feature engineering_, escalonamento, transformação de variáveis e regularização, com foco em preparar dados adequadamente para modelos de aprendizado de máquina.

## Conteúdos abordados

- Escalares, vetores e espaços vetoriais em Machine Learning
- Discretização de variáveis contínuas
  - Bins fixos
  - Bins variáveis (quantização)
- Transformações de potência com `PowerTransformer` (Yeo-Johnson)
- Normalização de dados
  - Min-Max Scaling
  - Standard Scaling
- Regressão Linear com regularização L2 (Ridge Regression)

## Tecnologias utilizadas

- Python
- Pandas
- Scikit-Learn
- [Dataset Palmer Penguins](https://allisonhorst.github.io/palmerpenguins/)

## Objetivo

Demonstrar, de forma prática, como diferentes técnicas de pré-processamento impactam a qualidade dos dados e o desempenho de modelos de Machine Learning.

Este diretório faz parte do repositório **data-prep-for-ml**, que reúne exercícios e exemplos práticos relacionados à preparação de dados para aprendizado de máquina.
