# Feature Engineering e Pré-processamento de Dados

Este repositório contém uma série de exercícios práticos sobre **Feature Engineering** e **pré-processamento de dados** aplicados a problemas de _Machine Learning_ e _Processamento de Linguagem Natural (NLP)_.

## Conteúdo

- Escalonamento de dados (StandardScaler)
- Normalização L1 e L2
- Implementação manual de técnicas de normalização
- Seleção de features:
  - Métodos de Filtragem
  - Métodos Wrapper (RFECV)
  - Métodos Embedded (conceito)
- Vetorização de texto:
  - Bag-of-Words
  - Bag-of-n-Grams
- Pré-processamento de texto:
  - Remoção de stopwords
  - Stemming
  - Lemmatization

## Datasets Utilizados

- Iris Dataset
- Breast Cancer Dataset
- Texto em língua portuguesa para tarefas de NLP

## Objetivo

O objetivo deste trabalho é compreender e aplicar técnicas fundamentais de **engenharia de atributos**, analisando seus impactos no desempenho e na representação dos dados para modelos de aprendizado de máquina.

## Tecnologias

- Python
- scikit-learn
- NLTK
- spaCy
- pandas
- numpy
