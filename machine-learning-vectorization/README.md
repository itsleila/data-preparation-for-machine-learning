# Feature Engineering e Vetorização de texto

Este repositório contém exercícios introdutórios de Machine Learning focados em **pré-processamento de dados**, **vetorização de texto**, _codificação de variáveis categóricas e modelos de classificação_.

## Conteúdo

- TF-IDF para vetorização de texto
- Regressão Logística para classificação de sentimentos
- Análise e visualização de coeficientes do modelo
- Técnicas de vetorização de variáveis categóricas:
- One-Hot Encoding
- Dummy Encoding
- Effect Encoding
- Feature Hashing
- Bin Counting (Count Encoding)
- Discussão sobre vantagens e desvantagens de cada técnica

## Os experimentos utilizam:

- O Large Movie Review Dataset (IMDB) para análise de sentimentos
- Um toy dataset para demonstrar diferentes técnicas de codificação categórica

## Objetivo

_Demonstrar, de forma prática, como diferentes técnicas de vetorização impactam a representação dos dados, a interpretabilidade dos modelos e o desempenho em tarefas de aprendizado de máquina._

## Tecnologias utilizadas

- Python
- Pandas
- Scikit-learn
- Statsmodels
- Mglearn
- Matplotlib
